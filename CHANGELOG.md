# تاریخچه تغییرات

تمام تغییرات مهم این پروژه در این فایل مستند می‌شود.

## [1.0.0] - 2024-02-08

### اضافه شده
- سیستم احراز هویت با JWT
- مدیریت کاربران با نقش‌های مختلف
- مدیریت تسک‌ها با وضعیت‌های مختلف
- مدیریت اسپرینت‌ها
- داشبورد با آمار و نمودار
- تقویم ماهانه برای نمایش تسک‌ها
- سیستم نوتیفیکیشن
- دستیار هوشمند AI
- طراحی UI/UX مدرن و فارسی
- API کامل RESTful
- مستندات جامع
- Docker support
- تست‌های واحد

### ویژگی‌های Backend
- Django 5.0 + DRF
- مدل‌های User، Task، Sprint، Notification، Comment
- API endpoints کامل
- JWT authentication
- Celery برای تسک‌های پس‌زمینه
- OpenAI integration

### ویژگی‌های Frontend
- React 18 + Vite
- Tailwind CSS
- React Router
- Context API برای state management
- کامپوننت‌های قابل استفاده مجدد
- طراحی ریسپانسیو
